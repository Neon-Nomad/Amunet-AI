## ğŸ¨ Design Philosophy

### "Keep the Movie, Don't Sell Magic"

Amunet AI is a **security engineer's power tool disguised as a sci-fi console**.

**Core Principles:**
1. **Terminal-First** - CLI is the hero, GUI supports it
2. **Explainability > Magic** - Every AI decision has a `why`
3. **Cinematic UX** - Hollywood-style output, real functionality
4. **Honesty** - No bullshit marketing claims
